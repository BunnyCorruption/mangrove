\subsubsection{Parameters for \codesnip{soundecology} Algorithms}
The following tables show the ranges and default values of the parameters for each algorithm in the \codesnip{soundecology} package. Note that these are taken into consideration in the implementation of this project\textquotesingle s Application Programming Interface.\par
