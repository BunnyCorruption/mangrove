\subsubsection{Get All Jobs}
Retrieve the details of all existing jobs. This also provides results for any any jobs whose \codesnip{status} is \codesnip{"finished"}.

\paragraph{GET} \mbox{}\\[\codeheaderspace]
\begin{htmlcode}
https://<base-url>/jobs
\end{htmlcode}
